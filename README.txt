Cipher Encryption & Decryption Application

Aplikasi ini adalah alat untuk melakukan enkripsi dan dekripsi teks menggunakan berbagai cipher, yaitu:
1. Vigenere Cipher
2. Playfair Cipher
3. Hill Cipher

Langkah-langkah Penggunaan
1. Masukkan Teks:
Anda bisa mengetik teks langsung di area input atau mengunggah file teks dengan mengklik tombol "Unggah File".

2. Masukkan Kunci:
Masukkan kunci enkripsi atau dekripsi sesuai dengan cipher yang dipilih.

3. Pilih Cipher:
Pilih salah satu cipher dari menu drop-down (Vigenere, Playfair, atau Hill).

4. Pilih Aksi:
Klik tombol "Enkripsi" untuk mengenkripsi teks, atau "Dekripsi" untuk mendekripsi teks.

5. Hasil:
Hasil dari proses enkripsi atau dekripsi akan ditampilkan di area output di bawah.
